# ProjectCatMapper GitHub Pages Root Site

One-page documentation hub for ProjectCatMapper.

## Live URL

- https://projectcatmapper.github.io/

## Includes

- CatMapper app links (SocioMap / ArchaMap)
- API quick links
- CatMapR package links and install snippet
- Contribution and governance references

## Deployment

Deployment is handled by GitHub Actions (`.github/workflows/pages.yml`).

In repository settings:

- Pages source should be set to **GitHub Actions**.
