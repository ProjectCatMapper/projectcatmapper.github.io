# Development Notes

## Stack

- Plain HTML/CSS/JavaScript.
- Deployment via GitHub Actions and GitHub Pages.

## Structure

- `index.html`: one-page hub.
- `styles.css`: visual style and responsive behavior.
- `script.js`: navigation and optional outbound click tracking hooks.
- `assets/`: logos and static media.

## Accessibility Baseline

- Use semantic section headings.
- Keep links descriptive.
- Preserve keyboard access for navigation.
